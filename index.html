Cougar Full Mission Simulator Middleware System Development

Rheinmetall Arabia | Defense & Simulation | Ongoing

Project Summary

Led the design, development, integration, and testing of the Middleware System for a Cougar Full Mission Simulator (FMS). The middleware acts as a core interface layer between the Host computer, Image Generator (IG), Computer-Generated Forces (CGF), Brief-Debrief System (BDS), and Instructor Operator Station (IOS). Key responsibilities included:

  • Implementing real-time CIGI (3.x) and DIS protocol conversion to drive the visual system with high-fidelity simulation data.
  • Ensuring sub-millisecond transport latency for IG communication via Start-of-Frame (SoF) and CIGI packets.
  • Developing interfaces for entity, articulation, and weapon DIS messages from CGF and converting them into CIGI to support dynamic battlefield rendering.
  • Coordinating middleware interactions with IOS, CGF, NPM Project Control, and BDS to support scenario control, playback, and synchronized simulation data display.
  • Delivering a robust architecture capable of supporting ownship and entity data across multiple subsystems including BDS and IG.

Technologies Used

  • C++
  • Boost (Interprocess, Asio)
  • Open DIS
  • CIGI Class Library (CCL)
  • PCAP/NPcap
  • Visual Studio
  • Shared memory
  • UDP sockets
  • Real-time systems
  • XML configuration

Key Achievements

  • Reduced IG communication latency to under 1 ms through efficient packet handling and memory management.
  • Designed and implemented a reliable shared memory model for real-time packet exchange between subsystems.
  • Overcame complex protocol conversion challenges by creating a flexible mapping system between DIS and CIGI.
  • Ensured visual consistency in IG by synchronizing lifecycle management for all DIS entities and IG updates.
  • Played a key role in end-to-end system validation and scenario-based testing for mission readiness.
